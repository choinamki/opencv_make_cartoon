# opencv_make_cartoon

c++, opencv 를 사용하여 카툰라이저를 만들었습니다.


opencv 는 4.1.1 버전을 사용했습니다.


입력받은 이미지를 만화처럼 바꿔줍니다.


string RAW_IMAGE_LINK 의 원하는 이미지 경로를 입력하면 사용할 수 있습니다.
